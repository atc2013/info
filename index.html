Общая информация

Курс автоматизации тестирования web-приложений предназначен для тех, кто хочет получить свой первый рабочий опыт в IT. В первую очередь мы ориентируемся на студентов 4-5 курсов, которые не только имеют желание связать свою карьеру с разработкой ПО, но и прикладывают для этого усилия. Занятия будут проходить на бесплатной основе в офисе компании Levi9 Ukraine.

Весь учебный период будет разбит на 2 этапа. На первом будут преподаваться основы тестирования ПО. На втором этапе начнется работа с автоматизацией web-приложений на Java + WebDriver. Таким образом в результате прохождения данного курса, участник будет обладать всеми необходимыми навыками для выполнения работы Junior Automation Test Devloper'a.

Первое занятие состоится 07.10.2013 в 19:00. В последующие 6 недель занятия будут проходить 3 раза в неделю (Пн, Ср, Пт) в 19:00.

Каждый претендент индивидуален и обладает личными человеческими качествами. Для эффективного обучения важны:

  • Ответственность
  • Настойчивость
  • Упорство
  • Пунктуальность

Существенным преимуществом будут знания в следующих технических областях:

  • Java
  • HTML
  • CSS
